## Setup guide

To connect to Nerdio, you'll need:

1. **Base URL**: Your Nerdio portal URL (e.g., `my-company-name.azurewebsites.net`)
2. **Tenant ID**: Your Microsoft Entra ID (Azure AD) tenant ID
3. **Client ID**: Your Azure AD application client ID
4. **Client Secret**: Your Azure AD application client secret

### Steps:

1. Enable REST API access in your Nerdio Manager portal
2. Create an Azure AD application for API access
3. Configure API permissions and generate client credentials
4. Note your Nerdio base URL and Azure tenant ID

For detailed setup instructions, see the [Nerdio REST API documentation](https://nmehelp.getnerdio.com/hc/en-us/articles/26124297445901-Getting-Started-with-REST-API-Integration).

## API gotchas

- Nerdio uses OAuth 2.0 authentication via Microsoft Entra ID (Azure AD)
- You must enable REST API access in your Nerdio Manager portal before using the API
- The base URL varies by deployment and can be found in your Nerdio portal URL
- API authentication requires both Azure AD credentials and your Nerdio base URL
